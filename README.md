# ABB Bank - Login Səhifəsi

ABB (Azərbaycan Beynəlxalq Bankı) karyera portalının giriş səhifəsinin HTML/CSS replikası.

## 📋 Məzmun

- `abb_login.html` — Tam işləyən login səhifəsi

## 🚀 İstifadə

Faylı brauzerinizdə açın:
```
abb_login.html
```

## 🎨 Xüsusiyyətlər

- ✅ Orijinal dizayna uyğun rənglər və layout
- ✅ Responsiv dizayn
- ✅ Şifrəni göstər/gizlət funksiyası
- ✅ Xalis HTML, CSS və JavaScript (heç bir kitabxana tələb olunmur)

## 🛠 Texnologiyalar

- HTML5
- CSS3
- Vanilla JavaScript

## 📸 Görünüş

Sol panel — mavi gradient fon, ABB loqosu və 3 üstünlük:
- Bərabər inkişaf imkanları
- Çevik iş mühiti  
- Yüksək komanda ruhu

Sağ panel — Email və şifrə ilə giriş formu.
